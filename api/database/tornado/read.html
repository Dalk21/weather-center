Database: https://the-weather.xyz/api/database/tornado

Introduction: This documentation provides the structure of a database of every EF5 tornado in JSON format. The database includes information on each tornado's name, description, date, fatalities, rating, location name, location zip code, and coordinates.
JSON Structure:
The JSON file is structured as follows:
The database is enclosed within a single object named "tornadoes".
The "tornadoes" object contains an array of objects that store information about individual tornadoes.
Each tornado object contains the following fields:
"name" (string): The name of the tornado.
"description" (string): A brief description of the tornado and its impact.
"date" (string): The date on which the tornado occurred.
"fatalities" (integer): The number of fatalities caused by the tornado.
"rating" (string): The EF scale rating of the tornado.
"locationName" (string): The name of the location where the tornado occurred.
"locationZipCode" (integer): The zip code of the location where the tornado occurred.
"coordinates" (array of two numbers): The latitude and longitude coordinates of the location where the tornado occurred.
