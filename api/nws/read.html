Documentation for Custom NWS API:
Introduction:

This API provides access to National Weather Service (NWS) weather forecasts and alerts, as well as radar images. The API is RESTful and provides data in JSON format. There are five endpoints that you can use to get weather information, which are:

Hourly forecast
Weekly forecast
Daily forecast
Weather alerts
Radar image
All endpoints require latitude and longitude coordinates to be provided in the URL as parameters.

API Base URL:
https://api.weather.gov

Endpoint 1: Hourly forecast
This endpoint provides hourly weather forecasts for the next few days at a given location.

URL:
/api/wx/hourly/:lat,:lon

HTTP Method:
GET

Parameters:

lat: The latitude of the location for which you want to retrieve weather information
lon: The longitude of the location for which you want to retrieve weather information
Response:
The API returns a JSON object containing the hourly forecast for the first period (next hour) of the location provided. The response includes the following fields:

temperature
windSpeed
windDirection
shortForecast
detailedForecast
Endpoint 2: Weekly forecast
This endpoint provides weekly weather forecasts for the next several days at a given location.

URL:
/api/wx/weekly/:lat,:lon

HTTP Method:
GET

Parameters:

lat: The latitude of the location for which you want to retrieve weather information
lon: The longitude of the location for which you want to retrieve weather information
Response:
The API returns a JSON object containing the weekly forecast for the location provided. The response includes the following fields for each day:

name
startTime
endTime
isDaytime
temperature
windSpeed
windDirection
shortForecast
detailedForecast
Endpoint 3: Daily forecast
This endpoint provides daily weather forecasts for the next several days at a given location.

URL:
/api/wx/daily/:lat,:lon

HTTP Method:
GET

Parameters:

lat: The latitude of the location for which you want to retrieve weather information
lon: The longitude of the location for which you want to retrieve weather information
Response:
The API returns a JSON object containing the daily forecast for the first period (next day) of the location provided. The response includes the following fields:

name
startTime
endTime
isDaytime
temperature
windSpeed
windDirection
shortForecast
detailedForecast
Endpoint 4: Weather alerts
This endpoint provides active weather alerts for a given location.

URL:
/api/wx/alerts/:lat,:lon

HTTP Method:
GET

Parameters:

lat: The latitude of the location for which you want to retrieve weather information
lon: The longitude of the location for which you want to retrieve weather information
Response:
The API returns a JSON object containing the active weather alerts for the location provided. The response includes the following fields for each alert:

event
headline
description
severity
urgency
certainty
effective
expires
instruction
Endpoint 5: Radar image
This endpoint provides a radar image for a given location.

URL:
/api/wx/radar/:lat,:lon

HTTP Method:
GET

Parameters:

lat: The latitude of the location for which you want to retrieve weather information
lon: The longitude of the location for which you want to retrieve weather information
Response:
The API returns a redirect to a radar image for the location provided. The image is a loop of the latest radar images from the National Weather Service.
